# ANSTEIA Website

**Tagline:** Thoughtfully automated. Beautifully designed.

A modern, bilingual (English/Spanish) website for ANSTEIA - a tech automation and creative services company.

## Tech Stack

- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS
- **Deployment:** Netlify
- **Languages:** English & Spanish

## Brand Colors

- Dusty Rose/Mauve: `#C5A5A5`
- Turquoise/Teal: `#4FA5A5`
- Slate Gray: `#5B6F7F`
- Sage/Mint: `#B5C5B5`
- Cream: `#F5F0E8`
- Deep Teal: `#3A7B7F`
- Soft Pink: `#E8C5D5`

## Development

### Install Dependencies
\`\`\`bash
npm install
\`\`\`

### Run Development Server
\`\`\`bash
npm run dev
\`\`\`

The site will be available at `http://localhost:5173`

### Build for Production
\`\`\`bash
npm run build
\`\`\`

### Preview Production Build
\`\`\`bash
npm run preview
\`\`\`
